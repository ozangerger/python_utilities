**calc_abs_time_diff()**

This function will print the absolute difference in seconds.

Input Format as:<br/>
date 1 as string<br/>
date 2 as string<br/>

Example input:<br/>
Sun 10 May 2015 13:54:36 -0700<br/>
Sun 10 May 2015 13:54:36 -0000<br/>

Example Output<br/>
25200
